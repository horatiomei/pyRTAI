def pyrtai::connector::Connector::getParameters (   self)

Asks the parameters to the XML-RPC server.

Returns:
Returns False if an error occurred. Otherwise returns an array of parameters. Each parameter is an hash with 5 elements: block_name, param_name, rows, cols, values. Values is a matrix of rows*cols elements.

Definition at line 192 of file connector.py.

def pyrtai::connector::Connector::getSignalStructure (   self)

Get the structure of the signals.

Returns:
Returns False if an error occurred or the target has no signals. Otherwise returns an array of signals. Every signal is an hash with 5 elements: number (int), name (string), traces_num (int), sample_time (double), traces_names (array).
